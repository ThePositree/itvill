## Скрипты:
1. Для скачивания зависимостей:
```
npm install
```
2. Для запуска в режиме разработки:
```
npm run dev
```
3. Для сборки:
```
npm run build
```
4. Для сборки в ZIP архив:
```
npm run zip
```
